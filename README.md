# [EN]

`This project is javascript game that developed for the BTU BLM0226 Web Based Programming class.`

## 💬 Scenario of the game
It has recently been revealed that coronaviruses have also come with people who have come to our planet from the world. And our healthcare professional, who devotes himself to this issue, does his best to destroy coronaviruses. Coronas deliver the orange coronavirus. In return, our healthcare worker destroys coronaviruses by sending a mask. If the low-level coronas are destroyed, the next level is continued. If the virus is infected by the corona, the public will be caught in the corona accordingly. Be careful save the people!

## 📸 Game screens && Game screenshots
The game contains 3 types of scenes as scenes. The screen on which the main game is played contains the game scenes, the screen that will welcome us when the corona is successfully destroyed, and finally the screen that will greet us when it fails. There is information about how to play the game and details of the project, which can be accessed from the entrance screen of the project.

# [TR]

`Bu proje BTU BLM0226 Web Tabanlı Programlama dersi için geliştirilmiş bir javascript oyunudur.`

## Oyunun Senaryosu
Yakın zamanlarda dünyadan gezegenimize gelen kişiler ile birlikte korona virüsleri de geldiği ortaya çıkmıştır. Ve kendini bu konuya adayan sağlık çalışanımız korona virüslerini imha etmek için elinden geleni ortaya koyar. Koronalar portakallı korona virüsü yollar. Sağlık çalışanımız buna karşılık maske yollayarak korona virüslerini imha eder. Düşük seviyede olan koronalar imha edilirse bir sonraki seviyeye geçilerek devam edilir. Eğer ki koronalar tarafından virüs kaparsa halkta buna bağlı olarak koronaya yakalanacaktır. Dikkatli ol halkı kurtar!

## Oyun Ekranları && Oyun Ekran Görüntüleri
Oyun sahnesi olarak 3 çeşitler sahne içeriyor. Ana oyunun oynalındığı ekran, koronaları başarılı bir şekilde imha edildiğinde bizi karşılayacak ekran, ve son olarak basarisiz olunduğunda bizi karsilicak ekran olmak üzere oyun sahnelerini içeriyor. Projenin giriş ekranından ulaşabilceğimiz oyunun nasıl oynancagini ve projenin detaylarını anlatılan bilgiler yer alıyor.
